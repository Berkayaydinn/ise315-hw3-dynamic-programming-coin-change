# ISE 315 – Homework 3

This repository contains my solution for Homework 3 of the course
ISE 315 – Analysis of Algorithms.

The homework focuses on solving the change-making problem using
Dynamic Programming.

## Contents
- main.py: Python implementation of the dynamic programming solution
  for the change-making problem.
- The code also includes a comparison with the greedy largest-first
  algorithm and a runtime experiment.
